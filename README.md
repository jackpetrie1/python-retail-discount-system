# Python Retail Discount System

This project is a Python program that simulates an online shop discount system.

## Features
- Accepts 5 products and prices from the user
- Sorts products from highest to lowest price
- Calculates total cost
- Applies discount by removing the cheapest item

## Skills Demonstrated
- Python programming
- Algorithms (sorting)
- Data structures (lists)
- Problem solving

## How to Run
1. Run the Python file
2. Enter product names and prices
3. View sorted products and total cost
